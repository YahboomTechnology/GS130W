# D-Robotics RDK GS130W/GS130WI binocular stereo global shutter camera module

# Introduction
The D-Robotics RDK GS130W is a high-performance binocular stereo global shutter camera module designed for precise and dynamic vision applications. Featuring dual SC132GS global shutter sensors, it supports synchronized binocular exposure and external triggering, delivering 1280×1080 resolution at up to 120 fps. With an 80mm baseline, high dynamic range (HDR), 40dB signal-to-noise ratio, and near-infrared enhancement, it is ideal for robotic vision, real-time motion tracking, and machine inspection. Fully compatible with RDK X5 and S100(P) development boards, the GS130W combines rugged semi-enclosed metal housing with reliable MIPI connectivity—ready for commercial deployment and scalable vision solutions.

# Features
* Global Shutter for Motion-Stable Imaging
  
Equipped with dual SC132GS global shutter sensors that support synchronous exposure and external triggering, eliminating motion blur and ensuring precise image capture in dynamic scenes—ideal for robotics and real-time monitoring.

* High-Speed, High-Resolution Vision

Delivers 1280×1080 resolution at 120 fps with high dynamic range (HDR) and 40dB signal-to-noise ratio, providing clear, detailed, and low-noise video for accurate depth perception and machine vision tasks.

* Optimized for RDK Ecosystem

Directly compatible with RDK X5 and RDK S100(P) developer boards via MIPI interface. Includes stereo depth algorithm resources and sample codes to accelerate integration and development in ROS-based applications.

* Rugged & Reliable Design

Semi-enclosed metal housing enhances durability, improves heat dissipation, and minimizes deformation for consistent performance in commercial and industrial environments.
